# Reviewer Teacher-Bank Evidence

This bundle is a compact evidence set for the teacher-bank distillation strategy.
It replaces hundreds of GB of original model checkpoints with their lightweight
prediction artifacts. Each teacher input is a legal prior prediction family, and
each ladder step is regenerated by distilling a selected bank of those prediction
families into a small ridge student.

The ladder demonstrates how the same code path can move from a conservative
baseline to the final selected family by changing one scalar:
`selected_teacher_weight`.

## Contents

- `teacher_bank_inputs/` - compact teacher prediction artifacts used as the bank.
- `example_submissions/` - generated submissions for each ladder step.
- `teacher_bank_ladder_metrics.csv` - retrospective/public-metric-style summary of the ladder.
- `configs/teacher_bank_ladder_steps.json` - step definitions and bank weights.
- `scripts/run_ladder_from_teacher_bank.py` - regenerates the ladder submissions from the compact bank.

## Reproducing the Ladder

Place this directory next to `generalization_submission_params/` and run:

```bash
./run_ladder_from_teacher_bank.sh
```

Or explicitly:

```bash
python scripts/run_ladder_from_teacher_bank.py \
  --package-root ../generalization_submission_params \
  --evidence-root . \
  --mpdd-root ../..
```

The generated outputs are written to `generated_submissions/`.

## Interpretation

The key axis is `selected_teacher_weight`. The early steps use only baseline and
train-derived prediction families. Later steps keep those families in the bank
while increasing the selected teacher's influence. This is a compact
teacher-bank calibration trace, not a claim that every original heavy checkpoint
is included in this bundle.
